# Growth Junction · Design Direction Review

Three contrasting design directions for the growthjunction.ae redesign.

## Live preview

Once deployed to Cloudflare Pages, the site lives at:
`https://growthjunction-designs.pages.dev/`

## Local preview

Clone the repo and open `index.html` in any browser. No build step.

```bash
git clone https://github.com/shabahatbokhari-ui/growthjunction-designs.git
cd growthjunction-designs
open index.html
```

## Files

| File | Direction | Tone |
|---|---|---|
| `index.html` | Review hub | Side-by-side comparison + how-to-use note |
| `01-editorial-premium.html` | Editorial Premium | Serif voice, long-form, HBR-style |
| `02-boutique-advisory.html` | Boutique Advisory | Clean sans, case-work led, procurement-friendly |
| `03-human-first.html` | Human First | Warm palette, faces, founder voice |

## How to review

1. Open `index.html` first
2. Click into each direction, spend 30 seconds, not 30 minutes
3. Pick the one that feels closest to "us", not the one most finished
4. Hybrids welcome — we can lift the hero of one onto the structure of another

## Status

Wireframe fidelity. Placeholder photography, illustrative copy, sample case studies. Not production designs. The point is to surface preference between directions, not to ship any of these as-is.
